## PostgreSQL
+ [Оглавление](../README.md)

+ Запустить сервер для пользователя MacOS

```
pg_ctl -D /usr/local/var/postgres start
```
+ Перезапустить сервер macOs
```
brew services restart postgres
```
+ Удалить каскадно
```
TRUNCATE some_table CASCADE;
```
